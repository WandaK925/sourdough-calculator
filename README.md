# Sourdough Starter Calculator

A simple, mobile-friendly calculator for sourdough starter feeding ratios.

## Features

- Two calculation modes:
  - **Feeding Mode**: Calculate how much flour and water to add to your existing starter
  - **Target Amount Mode**: Calculate ingredient amounts needed for a specific total amount of starter
- Customizable feeding ratios (default 1:4:4)
- Jar weight reference for quick lookups
- Works completely offline once loaded
- Mobile-optimized interface

- **Feeding Ratio**: Adjust the starter:flour:water ratio (default 1:4:4)
